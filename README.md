# Requerimientos previos

Python3, pytest


# Como usar main.py

Pide por pantalla un número binario al usuario y calcula su equivalente en decimal. Para salir, se debe escribir exit 


# Como usar lista.py

Pide por pantalla un número y indica si este se encuentra en la lista o si se encuentra entre los valores de la lista


# Cómo ejecutar

En el directorio src, ejecutamos python main.py o lista.py
Si estamos en el directorio raiz del proyecto, será python .\src\main.py o .\src\lista.py
